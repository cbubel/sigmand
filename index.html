Signature Manager

Quickly create and manage custom HTML signatures for use in Apple Mail. Inline CSS is easily visualized in a live-preview that guides design along the way.

Get Notified About Release

Why use Signature Manager?

The Apple Mail application does not provide native functionality to create custom HTML signatures - only plain text. If you want your email signatures to have images, links to sites, custom styles, and more, then the Signature Manager is for you.

Create

Use the built-in HTML editor to write custom code.

Assign

Easily assign signatures for use with Apple Mail accounts.

Preview

View a live-preview of your design as you code.

Delete

Quickly delete signatures that are no longer needed.
